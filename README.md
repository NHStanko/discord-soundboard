# discord-soundboard
Discord Soundboard Bot

## To Run - Development
1. [volta](https://volta.sh/) install node
2. npm install -D
3. Create .env file in main directory with BOT_TOKEN, GUILD_ID, and NODE_ENV=development
4. npm run build
5. npm run start
