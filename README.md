# discord-soundboard
Discord Soundboard Bot

## To Run - Development
1. [volta](https://volta.sh/) install node
1. npm install -D
1. Add token and guild ID to `config/token.env`
1. [Optional] Add additional environment overrides to `.env`
1. npm run build
1. npm run start
